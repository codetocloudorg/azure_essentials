Course description

Microsoft Azure expert Kevin Evans introduces you to a host of core Azure services, with a practical emphasis on how these services fit together in real-world environments. You’ll explore the foundational concepts of cloud computing and the Azure platform, including compute resources, storage options, networking, data services, and container orchestration.

Throughout the course, you’ll get hands-on experience using the Azure Portal and CLI, deploying virtual machines, configuring networking and storage, running Kubernetes workloads on AKS, and exploring container workflows with Azure Container Registry and container services. You’ll also work with data services such as Cosmos DB, get an introduction to Microsoft Fabric for analytics, and build simple automation and AI-driven applications using Azure Functions, Logic Apps, and Azure AI Foundry. In two days, you’ll acquire foundational skills that you can apply to cloud-based projects or roles involving Azure administration, application hosting, or cloud solution support.

Prerequisites
A free Microsoft Azure account (the entire course can be completed using free-tier SKUs), or access to the Azure Sandbox on the O’Reilly platform (Not all subscribers have access. Please check before the class.)
A review of the Code to Cloud platform engineering repo
The following VSCode extensions installed on your machine: HashiCorp Terraform, Azure Tools, Kubernetes, Azure Logic Apps, Red Hat YAML extension, and WSL
Azure CLI, Windows Subsystem for Linux, Azure Storage Explorer, and KubeCTL installed on your machine
A free Draw.io account set up and installed locally (optional)
A basic understanding of computer systems and networking concepts (IP addresses, DNS, firewalls)
Familiarity with operating systems, especially Windows and Linux fundamentals
Some basic familiarity with Kubernetes and containers
Some experience with scripting or command-line tools (PowerShell, Bash, or similar)
A general understanding of software development or application deployment concepts (helpful but not required)
